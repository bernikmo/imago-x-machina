# Imago x Machina

A growing collection of creative coding studies built with Processing.

Imago x Machina explores generative systems through small visual experiments, reconstructed artworks, drawing machines, and evolving sketches.
The focus is not only on finished images, but on the process behind them: structure, variation, randomness, perception, and transformation.

Each study is developed step by step.
Instead of isolated code snippets, the repository documents the evolution of an idea through complete sketches, notes, observations, and visual results.

Topics include:

* generative art studies
* algorithmic drawing systems
* order and disorder
* procedural variation
* grids, particles, textures, and noise
* image-based experiments
* creative coding workflows in Processing

The project begins with a reconstruction study of "Schotter" by Georg Nees and will continue to grow as an open visual notebook of experiments and discoveries.

Built with Processing / Java.
